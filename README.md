<h2 id="build-locally"><a href="#table-of-contents">Build Locally</a></h2>

1.  Prepare the `.env` file:

    ```
    SPONSORKIT_PATREON_TOKEN=<...>
    SPONSORKIT_AFDIAN_USER_ID=<...>
    SPONSORKIT_AFDIAN_TOKEN=<...>
    ```

2.  Install dependencies and build:

    ```bash
    npx pnpm install
    npm run build
    ```
